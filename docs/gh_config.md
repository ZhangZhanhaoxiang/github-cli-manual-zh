# gh config

显示或更改 gh 的配置设置。

当前接受的设置：

- git_protocol：用于 git 克隆和推送操作的协议（默认：“https”）
- editor：用于编写文本的文本编辑器程序
- prompt：在终端中切换交互式提示（默认：“已启用”）
- pager：向其发送标准输出的终端寻呼机程序
- http_unix_socket：通过其建立 HTTP 连接的 Unix socket 的路径
- browser：用于打开 URL 的 web 浏览器
