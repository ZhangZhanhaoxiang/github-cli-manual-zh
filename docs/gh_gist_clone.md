# gh gist clone

```
gh gist clone <gist> [<directory>] [-- <gitflags>...]
```

在本地克隆 GitHub gist。

gist 可以以下列任一格式作为参数提供：
- 按 ID，例如 5b0e0062eb8e9654adad7bb1d81cc75f
- 按 URL，例如“https://gist.github.com/OWNER/5b0e0062eb8e9654adad7bb1d81cc75f”

通过在“--”之后列出其他“git clone”标志来传递它们。
