- [gh issue status](/gh_issue_status)

## 选项

`-R`, `--repo <[HOST/]OWNER/REPO>`

使用 [HOST/]OWNER/REPO 格式选择另一个仓库

## 示例

```
$ gh issue list
$ gh issue create --label bug
$ gh issue view 123 --web
```
