# gh issue close

关闭 issue。

```
gh issue close {<number> | <url>} [flags]
```

## 选项

`-c`, `--comment <string>`

发表一条结束评论（closing comment）

`-r`, `--reason <string>`

关闭原因：\{completed|not planned\}

## 从父命令继承的选项

`-R`, `--repo <[HOST/]OWNER/REPO>`

使用 [HOST/]OWNER/REPO 格式选择另一个仓库
