# gh workflow list

```
gh workflow list [flags]
```

列出 workflow 文件，默认情况下隐藏禁用的 workflow。

## 选项

`-a`, `--all`

显示所有 workflow，包括禁用的 workflow

`-L`, `--limit <int>`

要获取的最大 workflow 数量

## 从父命令继承的选项

`-R`, `--repo <[HOST/]OWNER/REPO>`

使用 [HOST/]OWNER/REPO 格式选择另一个仓库
